Dividewise

Dividewise is my clone of the bill-splitting website Splitwise! I wrote it using the Ruby on Rails MVC framework and PostgreSQL as the database. The frontend was a work of React, Redux, Styled Components, CSS and HTML. You can find a Google© authentication API call inside the Rails controller with a Google login page for OAuth! I especially enjoyed adding tons of authenticating logic to make sure only people with the proper access could see and change bills because I believe that is important when it comes to pecuniary concerns.

Typing Fantasy

Typing Fantasy is a typing game that uses Javascript, Canvas, CSS, and the principles of OOP. I fetched from Google©Translation API to voice the words that appear above enemies, all of whom you can 'type to attack'. Word targets were detected as a partial-or-full hit by a trie data structure. Words per minute is tracked as well as a player's health. Additionally the game logic was divided into a Chinese and English version and there is logic to load only the dictionary needed (to save a player's bandwidth!). High scores are tracked in a in a json file to fit the interchange use-case. Keeping track of rendered frames and "layers" of what was on top of what was a real joy for me in this project. Be sure to try it out!

  • Show off your typing skills!

Bone Throw

Bone Throw is a team-built dogwalk-tracking site is a fullstack application that leverages MongoDB, Express, React, and Node (the MERN stack). It showcases the judicious use of modern in-browser functionality like Local Storage to keep track of sensitive info like a dog's weight in users' browsers rather than in a traditional database. Furthermore, we created a Python Django API that fetched from a Yahoo stocks API and transformed that stock data into a continuous wave with another physics API before then sending it off to Bone Throw for the user to see as a beautifully rendered Charts.js data chart! Additionally, data 'streams' like that of many user-input walk durations were leveraged by the MongoDB database and shown off to the world to keep users engaged and informed of a walk recommendation.
